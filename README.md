# UBBike | Propuesta Web

Landing page en Astro para presentar la propuesta de control y trazabilidad del servicio de bicicleteros UBB.

## Requisitos

- Node.js 22.12 o superior
- npm 10 o superior
- Git

## Instalacion

```bash
git clone https://github.com/SebastianPinto2424/Propuesta-pagina-proyect.github.io.git
cd Propuesta-pagina-proyect.github.io/pink-proxima
npm install
```

## Ejecucion local

```bash
npm run dev
```

Abrir en navegador:

```text
http://localhost:4321/Propuesta-pagina-proyect.github.io/
```

## Comandos utiles

```bash
npm run dev      # desarrollo
npm run build    # build de produccion
npm run preview  # vista previa de build
```
